# <a name="about-project">My Bloging App</a>

**My Bloging App** is a simple bloging app that supports all CRUD operations. With this app, a user can create a blog, reading it from the database, update and delete it. A user can also comment and like a blog post.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 gem install rails
```


### Setup

Clone this repository to your desired folder:

```sh
  git@github.com:Mukumbuta/My_Bloging_App.git
  
  cd My_Bloging_App
  
```

### Install

Install this project with:

```sh
  cd rails-blog-app
  gem install
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```
### Run tests

To run tests, run the following command:

```sh
  bin/rails test test/models/article_test.rb
```
